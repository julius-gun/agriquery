<h2>To be done before spreading</h2>
<h2>Working width</h2>
<p>The working width must be selected to set the spreader. This could for instance be the distance between the sprayer tramlines.</p>
<p>In the spreading chart book, via the online spreading chart service on the website or via the free application (App) for mobile phones with an iOs and/or Android operating system, look for the page that shows the desired working width of the fertiliser type with the most similar properties.</p>
<p>Various types of fertiliser have a limited maximum working width. This is due to their physical qualities whereby they do not spread evenly over large working widths. If a working width is not given in the spreading charts, the relevant type of fertiliser cannot be spread evenly over that width.</p>
<p>If you want to spread with a working width not given in the spreading charts, use the settings for the working width closest to the selected working width. For example: the settings given in the spreading chart for a required working width of 25 metres could, for instance, be approximated to 24 metres. Enter the actual working width into the control system so that the selected application rate is correctly distributed.</p>
<p>The working width is adjusted with:</p>
<ul>
<li>· the correct connection point for the coupling shaft with the PTO speed and</li>
<li>· the discharge point.</li>
</ul>
<p>The working width of the spreader is also determined by the chosen connection point for the coupling shaft on the central gearbox. Which terminal (turtle or hare) you need to use is listed in the (online) spreading chart and depends on the desired working width.</p>
<ul>
<li> Switch the tractor off, remove the key from the ignition and engage the parking brake of the tractor.</li>
<li> Detach the coupling shaft from the machine.</li>
<li> Attach the coupling shaft to the correct attachment point.</li>
</ul>

<h2>Discharge point</h2>
<h2>Inclination</h2>
<figure><figcaption>Angle indicator</figcaption></figure>
<h2>Modification during GEOCONTROL use</h2>
<h2>PTO shaft rpm</h2>
<figure></figure>
<p>It is necessary to correctly set the discharge point in order to obtain a good spreading pattern and the correct working width. The point at which the granules or seeds come onto the spreading disc can be changed by setting the discharge point. The position of the discharge point combined with the position of the coupling shaft on the central gearbox, the PTO speed and the flowrate determines the spreading pattern.</p>
<p>The discharge point can be set between positions C to U. The correct settings are determined on the basis of the selected application rate and the required driving speed. The position that would most likely provide the best spreading pattern is given in the third column, Discharge Point, of the spreading chart.</p>
<p>Always check the accuracy of the discharge point by conducting spreading pattern tests (tray test).</p>
<ul>
<li> see instructions in paragraph »Spreading pattern check (tray test)« on page 76.</li>
</ul>
<p>In the (online) spreading chart, look for the Hoek spreader symbol to get advice on the angle of your spreader. The spreader is normally in the horizontal position. It may be necessary to adjust the spreader's angle with large working widths or with top dressing.</p>
<p>You can set the correct angle by adjusting the top link. There is an indicator on the spreader which displays the angle of the spreader.</p>
<ul>
<li>TIP Set the spreader angle before you fill the hopper. It is easier to adjust the top link if the hopper is empty.</li>
</ul>
<p>Check the position again after the hopper has been filled!</p>
<p>If the inclination is at 4 o of 8 o, the spreader area slides further behind the spreader.</p>
<ul>
<li> In the machine data screen, increase the length B by 10% or 20% at 4 o or 8 o.</li>
<li> Please refer to the software instruction manual for the procedure.</li>
</ul>
<p>The (online) spreading chart shows the rpm of the spreading discs as well as the rpm of the PTO shaft. The PTO speed is given, as well as the connection point to be used for the machine coupling shaft. You have to calculate the speed of the spreader discs. The actual RPM of the spreading discs is displayed on the control box screen.</p>

<h2>To be done before spreading</h2>
<h2>Application rate</h2>
<h2>Constant application rate</h2>
<figure></figure>
<figure></figure>
<figure></figure>
<p>The control system can drive the spreader in three different ways:</p>
<ul>
<li>· distribution that resembles the settings for the preset application rate must be determined under changing circumstances through automatic calibration and driving speed correction,</li>
<li> see the paragraph below,»Constant application rate«.</li>
</ul>
<p>with similar or lower application rates (< 40 kg/min) the fine application kit should be used,</p>
<ul>
<li> See paragraph »Fine application« on page 67.</li>
<li>· with very low application rates on very uneven terrain, preset application rates must be checked manually to ensure that the correct dosing setting is being distributed, without automatic calibration but with driving speed correction,</li>
<li>· under changing circumstances location specific application rate driven by an external signal, through automatic calibration and driving speed correction,</li>
<li> See paragraph »Variable, externally driven application rate« on page 72.</li>
</ul>
<p>The control system can spread an equal amount of fertiliser over the entire field. In doing so, the control system corrects changes in driving speed, while maintaining the option to adjust the application rate manually for over- or underdosing.</p>
<p>The control system can search for the correct dosing setting with the preset application rate through automatic calibration. The disadvantage is that the application rate for the second automatic calibration (= approx. 2 minutes) could differ markedly from the selected application rate. It is therefore strongly recommended that the starting values be taken from the spreading charts. This can be done in the 'Curve Calibration' screen. The control system calculates the discharge curve from the starting values. The control system corrects the spreader for changing circumstances, based on this curve, e.g. as a result of changes in driving speed.</p>
<ul>
<li> See also paragraph »The ISO control system« on page 30.</li>
</ul>
<p>Look for the most comparable fertiliser type and the correct settings for the selected working width in the (online) spreading chart:</p>
<ul>
<li> Read the setting of the dosing in the Dosing Setting column that goes with the selected application rate and driving speed.</li>
<li> The corresponding flowrate is given in the Kg/min column.</li>
<li> The given values are to be entered in the control system.</li>
</ul>
<p>If the selected application rate is not given in the spreading charts, use the settings that are the closest to the selected application rate. If the selected driving speed is not given, use the settings closest to the selected driving speed.</p>

<h2>Fine application</h2>
<figure><figcaption>Fine application handle Normal dosing setting</figcaption></figure>
<figure></figure>
<figure><figcaption>Fine application handle Fine application setting</figcaption></figure>
<p>All three dosing openings normally open uniformly with the selected application rate and the corresponding flowrate that you set. With a low application rate and/or a low driving speed the dosing openings are only opened a slight bit so that very little fertiliser would stream through the dosing openings of the spreader. This could easily cause blockage, which would lead to a bad spreading pattern and possible damage the crops.</p>
<p>We therefore advise that you make use of the fine application if the flowrate is less than 40 kg/min.</p>
<p>The flowrate with a specific working width, application rate and driving speed is given in the fourth column of the spreading chart.</p>
<p>If very low application rates (less than 25 kg/ha) are to be distributed, e.g. only a few kilograms per hectare, the control system cannot perform automatic calibrations. The distributed weight is too small to accurately adjust the application rate.</p>
<p>Close 2 of the 3 dosing openings to prevent blockages. The remaining opening must be further opened as the application rate with the constant flowrate would be three times as much.</p>
<p>The handle to close 2 of the 3 fine dosing openings is located at the outside of the spreading disc, above a similar handle for the discharge point. You will find the indications on top of the fine application handle for the number of dosing openings: 3 open holes for standard dosing and 1 open hole and 2 thick holes for fine application.</p>
<ul>
<li> Switch the tractor off, remove the key from the ignition and engage the parking brake of the tractor.</li>
<li> Pull the hand-grip of the fine application lever out.</li>
<li> Set the fine application handle until the arrow on the hand-grip indicates the desired position.</li>
<li> Let go of the hand-grip and check if the fine application handle is locked.</li>
<li> Set the fine application lever on the other spreading disc in the same way to the same position.</li>
</ul>

<h2>Setting the spreader</h2>
<p>The dosing setting to be set can be:</p>
<ul>
<li>· Read from the relevant spreading chart when you want to spread at a lower application rate using the fine application kit and the symbol of the Fine application kit is indicated in the spreading chart. This is indicated by means of 1 open hole (dosing opening).</li>
<li>· Determine an application rate not given as described in paragraph »Discharge point«from page 65.</li>
<li>· To be calculated using the relevant spreading chart when you wish to spread at a lower application rate with the fine application kit, and the Fine application kit symbol is indicated in the spreading chart; it is indicated using 3 open holes (dosage opening).</li>
</ul>
<h2>Calculation</h2>
<p>As 2 of the 3 dosing openings are closed the flowrate of the remaining dosing opening should be three times as much in order to obtain an even application rate.</p>
<ul>
<li> Read the Dosing Setting, Discharge Point and Flowrate from the spreading chart. Calculate the flowrate with the formula given below as required.</li>
</ul>
<p>If the selected application rate or driving speed is not indicated in the spreading chart the required flowrate can also be calculated as follows:</p>
<p>flowrate required (kg/min) =</p>
<p>Working width (m) x Driving speed (km/h) x Application rate (kg/ha)</p>
<p>600</p>
<p>Use the letter given for setting the discharge point in the line of the spreading chart that most closely resembling the calculated flowrate.</p>
<ul>
<li> Multiply the flowrate (given in the Kg/min column or calculated) with a factor of 3.</li>
<li> Select the closest flowrate for the whole spreader in the Kg/min column.</li>
<li> Read off the relevant dosing setting or estimate it as closely as possible.</li>
</ul>

<figure><figcaption>NOTE  Do not use the discharge point indicated there!</figcaption></figure>

<h2>Manual calibration test</h2>
<figure></figure>
<figure><figcaption>support for the calibration container</figcaption></figure>
<p>support for the calibration</p>
<figure></figure>
<p>S</p>
<figure><figcaption>Lock nut</figcaption></figure>
<p>NOTE</p>
<p>Do not empty the calibration container!</p>
<p>Rinsing the calibration container greatly improves the accuracy the calibration test. All parts of the calibration container where fertiliser, seeds or granules could remain would now be filled.</p>
<p>The calibration test can now start.</p>
<ul>
<li> Please refer to the software instruction manual for the procedure.</li>
</ul>
<h2>To be done before spreading</h2>
<h2>PTO shaft rotates after disengaging</h2>
<p>After switching off the PTO shaft, danger could arise as the machine will continue running because of inertia. Keep a safe distance to the machine until the moving parts have come to a complete stop. Approaching the machine while it is rotating can cause serious or deadly injuries.</p>
<p>You can check with a manual calibration test if the preset dosing setting reaches the selected flowrate.</p>
<p>The calibration test is normally performed on the left spreading disc. Perform the calibration test with a filled hopper.  See also paragraph »Filling« on page 73.</p>
<ul>
<li> Switch the tractor off, remove the key from the ignition and engage the parking brake of the tractor.</li>
<li> Disconnect the right spreading disc by sliding the connector to the right. Secure the connector with a lock clip.</li>
<li> Remove the vanes from the left spreading disc.</li>
<li> Connect the coupling shaft to the coupling point on the machine shown in the relevant spreading chart.</li>
<li> See also paragraph »Coupling shaft connection point« on page 64.</li>
<li> Mount the calibration container carefully on the left spreading disc.</li>
<li> Set the calibration container with the settings knob so that the spreading disc does not touch the calibration container.</li>
<li> Weigh a large trough and place it under the discharging point of the calibration container.</li>
<li> Start the tractor again.</li>
<li> Close all the doors and windows in the cabin.</li>
<li> Switch the control box on.</li>
<li> Switch the right spreading disc off.</li>
<li> See the software instruction manual.</li>
<li> Engage the PTO shaft.</li>
<li> Open the left dosing unit.</li>
<li> Allow the spreader to spread for 5 seconds in order to 'rinse' the calibration container.</li>
<li> Close the dosing unit.</li>
<li> Switch the PTO shaft and the tractor off, remove the key from the ignition, empty the trough and place it back under the discharge point of the calibration container.</li>
</ul>

<h2>Filling</h2>
<figure></figure>
<h2>Avoid contact with the fertiliser and other material</h2>
<p>You could come into contact with materials when filling the machine. Always wear protective clothing, such as safety gloves, shoes and goggles. Read and adhere to the additional safety instructions and the of the material producer. Contact with the materials could lead to injury. Contact your physician in such an event!</p>
<h2>Chemical reactions caused by mixing fertilisers</h2>
<p>Mixing different types of fertiliser could lead to a chemical reaction whereby poisonous vapours, liquids or other materials could come about. Possible danger of explosion! This could lead to personal injury and damage to the crop and machine.</p>
<p>Only mix different types of fertiliser if allowed by the manufacturer(s).</p>
<p>Chemical reactions could lead to damage to the machines and personal injury.</p>
<h2>Read the safety instructions provided by the supplier</h2>
<p>Read the safety instructions provided by the supplier and apply that. Protective clothing should be worn when necessary.</p>
<p>Disregarding the safety instructions could result in personal injury and damage to the machine.</p>
<h2>Use proper tools</h2>
<p>Great volumes of material is often handled during the filling process. Use proper tools such as lifting devices when moving and filling the machine. Keep legislation concerning working conditions in mind. Work carefully and responsibly. Inferior tools can cause serious inju-</p>
<p>ries.</p>
<h2>Only fill mounted machines</h2>
<p>Never place a machine in the ground while filling it. Always keep it in the power lift of the tractor. The machine is designed to be loaded while in a suspended state. Never park a filled machine on the ground. Filling the machine while it is on the ground could lead to damage.</p>
<h2>Work with a closed cabin</h2>
<p>Fertiliser dust can blow indoors through open windows, doors or other openings. Moreover, the sound level is higher in an open cabin. It is therefore recommended to always work with a closed cabin with active carbon filters in the air inlets. Fertiliser dust and a high sound level can cause physical damage.</p>

<p>Most types of fertiliser attract water and can thereby form large chunks. These chunks can block the spreader or distort the spreading pattern.</p>
<ul>
<li>· When working under moist conditions, use a hopper cover that closes well.</li>
<li>· Do not spread without a hopper cover under moist conditions.</li>
<li>· Never leave the fertiliser for a long period of time in the spreader. Chunks can also be formed through moisture in the air.</li>
<li>· Always clean the spreader well after use, including the dosing openings and the discharge bushings.</li>
</ul>
<p>You can fill the spreader to the desired levels after it has been put in place. It is recommended that the machine be loaded while it is lifted 10 cm above the soil. If no activities need to be performed on top of the hopper, the height can be adjusted with the automatic filling system.</p>
<p>A load platform must be used if the distance between the hopper rim and the ground is more than 1250 mm while filling manually or with large bags. The filling height is given in</p>
<p> paragraph »Technical specifications« on page 36.</p>
<ul>
<li> If a fixed loading platform is used, drive the machine to the platform. The distance between the machine and the platform must be a maximum of 200 mm.</li>
<li> Lower the machine at the desired level. The machine must not rest on the ground, but rather be lifted a minimum distance of 10 cm!</li>
<li> Check if the dosing units are closed. Close them if necessary by pressing the stop key on the main screen.</li>
</ul>
<p>The position of the dosing units is displayed in the main screen on the control box of the control system.</p>
<ul>
<li> Switch the electronic control system off.</li>
<li> Switch the tractor off and remove the keys from the ignition.</li>
<li> Engage the parking brake of the tractor.</li>
<li> Close all the doors and windows in the cabin.</li>
<li> Place wheel chocks in front of the wheels of the tractor.</li>
<li> If a mobile loading platform is being used, place the loading platform at the spreader so that there would be a maximum distance of 200 mm between the two.</li>
<li> Open the hopper cover.</li>
<li> Check if the sieve is closed and secured.</li>
<li> Fill the machine to the desired level with the correct tools. Fill the hopper halves to the same level so that the both of them would run empty at the same time.</li>
</ul>

<h2>To be done before spreading</h2>
<h2>Spreading pattern check (tray test)</h2>
<h2>Avoid contact with the fertiliser and other material</h2>
<p>You will most likely get into contact with the fertiliser and other material you may wish to spread while filling, adjusting and testing the spreader. Wear protective clothing and gloves, read the safety instructions provided by the supplier and follow their directions. Contact with these materials could cause physical harm.</p>
<h2>Be careful that nothing gets jammed</h2>
<p>It is easy for parts to get jammed while adjusting the spreader. Work carefully:</p>
<ul>
<li>· switch the tractor off and remove the key from the ignition,</li>
<li>· engage the parking brake of the tractor,</li>
<li>· wait for all machine parts to come to a standstill before you start with the adjustments,</li>
<li>· carefully follow the instructions given.</li>
</ul>
<p>Friction with these materials could cause physical injury.</p>
<h2>The test should be performed on your own. Operate the machine on your own!</h2>
<p>Dangerous situations for other people could easily occur when the machine is turning and the test conducted. Contact with these materials could cause physical injury.</p>
<h2>Protecting the environment</h2>
<p>Spreading a too high dose of materials outside of the field edge, especially on water surface, creates a heavy burden on the environment. Protect the environment by applying a correct application rate in a correct working manner.</p>
<h2>PTO shaft rotates after disengaging</h2>
<p>After switching off the PTO shaft, danger could arise as the machine will continue running because of inertia. Keep a safe distance to the machine until the moving parts have come to a complete stop. Approaching the machine while it is rotating can cause serious or deadly injuries.</p>
<h2>Work with a closed cabin</h2>
<p>Fertiliser dust can blow indoors through open windows, doors or other openings. Moreover, the sound level is higher in an open cabin. It is therefore recommended to always work with a closed cabin with active carbon filters in the air inlets. Fertiliser dust and a high sound level can cause physical damage.</p>
<p>A tray test checks the distribution of the fertiliser as the spreader distributes it. The results of this test will indicate if you can improve the settings of the spreader, thereby improving the distribution of fertiliser and limiting loss.</p>
<p>Tray tests can be performed by specialised companies, but it is also possible to perform this test on your own. You will have to purchase some materials for it, however.</p>
<p>We recommend that you request a specialist company to perform the tray test as they can also provide you with advice and answer your questions.</p>

<p>INDICATION</p>
<h2>When to perform the tray test:</h2>
<ul>
<li>· When using a fertiliser type that is not in the spreading chart book, via the online spreading chart service on the website or via the free application (App) for mobile phones with an iOs and/or Android operating system.</li>
<li>· When using a fertiliser type of which the distribution and density deviate from the values indicated in the spreading chart book, via the online spreading chart service on the website or via the free application (App) for mobile phones with an iOs and/or Android operating system.</li>
<li>· When using fertiliser of moderate quality.</li>
<li>· Test should be carried out annually on the left and the right side to check the equal distribution of both spreading discs.</li>
<li>· After each change to the spreading mechanism of the spreader or when parts are replaced.</li>
</ul>
<p>Weather conditions have a great influence on the spreading pattern while spreading, as well as while performing the tray test. The tray test should be performed with dry weather, low humidity and with a maximum wind speed of 3 m/s (wind force of 2).</p>
<h2>Material required</h2>
<ul>
<li>· measuring tape or ruler,</li>
<li>· spirit level to set the troughs horizontal,</li>
<li>· 7 troughs,</li>
<li>· 7 graduated tubes,</li>
<li>· 1 funnel,</li>
<li>· notebook, pen and calculator,</li>
<li>· this manual.</li>
<li>· the instruction manual of the software.</li>
</ul>
<h2>General instructions</h2>
<ul>
<li> Check the preset dosing setting and the discharge point.</li>
<li> Check the height (75 cm above the troughs) and the position (angle) of the spreader.</li>
<li> Check the rpm of the PTO shaft and the spreader discs.</li>
<li> Always place the buckets horizontally on the floor.</li>
<li> Place the buckets at a maximum distance of 4 metres. Provide additional widths for testing widths of more than 24 meters. Also, you can first test part of the working width and then empty the trays and distribute them over the rest of the working width.</li>
<li> Do not use the spreader while it is raining or with high humidity. This will cause the fertiliser to cling to the troughs, which would not provide the desired results.</li>
<li> Do not spread with wind speeds of more than 3 m/s (wind force of 2).</li>

<h2>To be done before spreading</h2>
<h2>Test the left and right spreading patterns</h2>
<p>INDICATION</p>
<figure></figure>
<p>The left and right spreading discs must spread the same volumes of fertiliser to ensure good distribution. Check this every year!</p>
<p>This test describes a working width of 24 metres. You are of course free to run the test for a different working width. Divide the buckets at equal distances along the working width.</p>
<h2>Operation</h2>
<ul>
<li> Switch the tractor off, remove the key from the ignition and engage the parking brake of the tractor.</li>
<li> Disconnect the left spreading disc by sliding the connector to he left. Secure the connector with a lock clip.</li>
<li> Search the correct discharge position to spread over 24 metres in the best comparable fertiliser, seeds or granules (online) spreading chart and set the handle accordingly.</li>
<li> See paragraph »Discharge point« on page 65 and the foreword of the spreading chart book.</li>
<li> Fit the coupling shaft to the correct attachment point of the central gearbox as indicated in the associated spreading chart.</li>
<li> See paragraph »Coupling shaft connection point« on page 64.</li>
<li> Check the height of the spreader (height of the spreading disc = 75 cm above the troughs).</li>
<li> Place the grids in the troughs.</li>
<li> Place trough 1 in the middle of the tractor tracks.</li>
<li> Place the other troughs 4 metres to the right of the tractor tracks.</li>
</ul>
<p>INDICATION Position the troughs horizontally with a spirit level!</p>

<h2>To be done before spreading</h2>
<ul>
<li> Start the tractor again.</li>
<li> Close all the doors and windows in the cabin.</li>
<li> Switch the control box on.</li>
<li> Enter the required application rate.</li>
<li> Enter the working width.</li>
</ul>
<p>It is recommended that an application rate of at least 250 kg/ha is entered with the selected driving speed.</p>
<ul>
<li> Switch on the left spreading disc.</li>
<li> Switch off the automatic calibration.</li>
<li> Engage the PTO shaft and set the RPM of the spreading discs as given in the relevant spreading chart.</li>
<li> Open the right dosing unit.</li>
<li> Drive over the first trough and stop only after you have driven at least 20 metres past this trough. This is because the spreader distributes roughly even to the back, front and to the side.</li>
<li> Close the dosing unit.</li>
<li> Empty the trough with a funnel into the graduated tubes and record the reading in your notebook.</li>
</ul>
<p>Repeat the above procedure, but disconnect the right spreading disc and switch the dosing unit of the right spreading disc off. Put the troughs on the left side of the tractor track.</p>
<p>Drive with the same speed and with the same PTO shaft rpm as with the first time!</p>

<li> Perform the »Checking and setting the discharge point setting mechanism« as described on page 136 and repeat the test.</li>

<h2>Full field spreading pattern test</h2>
<figure></figure>
<p>If the spreader is correctly set and the tramlines are set at the correct distance from each other you would achieve a good distribution of fertiliser over the land. This will only be achieved if the spreader spreads with the correct spreading pattern.</p>
<p>This is then the reason why you have to test the spreading pattern and adjust it as required.</p>
<ul>
<li> See also paragraph »Full field spreading pattern« on page 28.</li>
</ul>
<h2>Operation</h2>
<ul>
<li> Switch the tractor off, remove the key from the ignition and engage the parking brake of the tractor.</li>
<li> Set the spreader as per the details given in the spreading chart for the selected working width (X metre) that most closely matches the fertiliser, seeds or granules (height of the spreading disc = 75 cm above the troughs).</li>
<li> Place the grids in the troughs.</li>
<li> Place trough 1 in the middle of the tractor tracks.</li>
</ul>
